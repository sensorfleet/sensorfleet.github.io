Downloader Instrument

Overview

Downloader is helper Instrument providing other Instruments the ability to download resources from outside the sensor (intra-/internet).

Some Instruments do not require active network connection for their operation, but can use outside resources for configuration. Using downloader Instrument to fetch those resources allows them to require only passive network connection while still be able to get the updated resources.

Downloader Instrument will handle fetching the resources from internet and placing them where the requesting Instrument can then read them. Using downloader Instrument also allows to separate internet access to single Instrument while still maintaining the ability for fetching resources when needed.

Downloader checks configured resources periodically fetching them only if the server indicates that the contents have been changed.

Required downloads for Instruments can be configured using the Fleet Management UI.

For operation, Downloader Instrument needs interface with network address, route to resources and configured name server. These can be configured using the Fleet Management UI.

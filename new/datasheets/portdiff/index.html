PortDiff Instrument

Overview

PortDiff Instrument is a tool for tracking the discoverable network software of assets in your network. The instrument will notify you using SensorFleet events when there are changes in the network. The scan is performed using industry standard methods for TCP/IP port scanning.

Events include:

  • TCP/UDP listen ports changes

  • Software fingerprint changes (e.g.Â software version, OS version) (*)

  • New/missing IPs

(*) Fingerprinting capability is limited to the most typical software and a very limited OS detection.

PortDiff instrument properties

Instrument developer SensorFleet Oy
Network access type Active
Required interfaces Monitored network (any with receive, send capabilities)
Instrument dependencies None
Data retention Besides the current active network status, the instrument stores no data. All history will be stored in events on the sensor, not in the instrument.
Management UI Yes
Performance Scanning will become incrementally slower when going for networks with over 256 hosts. Maximum recommended network size 5000 hosts.
